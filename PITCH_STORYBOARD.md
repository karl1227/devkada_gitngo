## PRODUCTION CHECKLIST

### Pre-Recording:
- [ ] Test all features work smoothly
- [ ] Prepare demo accounts (parent and teacher)
- [ ] Set up screen recording software
- [ ] Prepare backup screenshots if live demo fails
- [ ] Test microphone and audio quality
- [ ] Ensure stable internet connection
- [ ] Close unnecessary applications
- [ ] Set up good lighting for face cam (if using)

### During Recording:
- [ ] Speak clearly and at moderate pace
- [ ] Pause between sections for editing
- [ ] Show features smoothly (no rushed movements)
- [ ] Highlight important UI elements
- [ ] Keep cursor movements natural
- [ ] Test live demo link works

### Post-Recording:
- [ ] Edit to ensure under 3 minutes
- [ ] Add transitions between scenes
- [ ] Add background music (optional, low volume)
- [ ] Add text overlays for key points
- [ ] Ensure audio is clear
- [ ] Export as MP4, less than 1GB
- [ ] Test video plays correctly
